Member Enumeration Documentation

enum Stencila::CilaParser::State

Alternative parsing states

Enumerator
sol 

Start of line state

elem 

Looking for element

In this state can move across into attrs

attrs 

Looking for element attributes

In this state the parser is looking for HTML element attribute syntax (e.g. [id="an-id"], #an-id, .a-class) including directives (e.g. text x) and ignoring whitepace. If no attribute is found then moves across to text state.

flags 

Looking for rendering flags (e.g. hash, index, off) some of which are only applied to directives:

  • hash
  • off and others which can be applied to both directives and normal elements
  • index
  • lock
  • included
text 

Text including shorthands and ilined elements

empha 

Within an emphasis section (e.g _this text is emphasised_)

strong 

Within a strong section (e.g *this text is strong*)

code 

Within a code section (e.g answer = 42)

asciimath 

Within an AsciiMath section (e.g |e = mc^2|)

tex 

Within a TeX/LaTeX section (e.g \(e = mc^2\))

embed 

Within an exec a style directive. Embedded code.
