Detailed Description

Note that boost::property_tree::ptree does not hold any type information and so outputs all values as text. See: 

            https://svn.boost.org/trac/boost/ticket/9721
            http://stackoverflow.com/questions/2855741/why-boost-property-tree-write-json-saves-everything-as-string-is-it-possible-to

Another option is to use JsonCpp for all this. At present, this was implemented using Boost only to reduce requirements for compiling structures.
