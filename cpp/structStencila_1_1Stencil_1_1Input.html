Detailed Description

An <input> element (e.g. <input name="answer" type="number" value="42"></input>)

input elements are used for setting variables in the context using untrusted user supplied data. Variables must be of a specified type. For trusted user content, the analogue of an <input> element is a set directive which takes a language expression (which may be any type).
