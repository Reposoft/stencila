Detailed Description

An execute (exec) directive (e.g. <pre data-exec="r,py">)

The text of the element is executed in the context if the context's type is listed in the data-exec attribute. If the context's type is not listed then the element will not be rendered (i.e. will not be executed).

This behaviour allows for polyglot stencils which have exec directives that are either polyglot (valid in more than one languages) or monoglot (valid in only one language) as required by similarities/differences in the language syntax e.g.

       m = 1
       c = 299792458
   
 e = m * c^2 
 e = m * pow(c,2)
