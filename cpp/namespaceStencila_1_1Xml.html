Detailed Description

Stencila's interface to Extensible Markup Language (XML).

The Stencila library currently uses pugixml as a backend for XML parsing and generation. We chose pugixml because it is fast and has XPath support. Documentation for pugixml is available here.

There is a brief, but very good, guide on how to choose a XML library here. It summarises many of the conclusions that we came to in choosing pugixml for Stencila.
