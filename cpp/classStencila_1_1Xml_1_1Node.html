Detailed Description

XML node

This class wraps pugi::xml_node to change the programming interace. It adds convienient methods for appending elements to a node e.g. elem.append("div",{{"class","greeting"},{"id","hello"}},"Hello world"). Where appropriate it reduces the length of method names (e.g next_sibling() becomes next()) and, to some extent, makes them consitent with the JQuery API (e.g. attribute("class") becomes attr("class")).

Sanitize the Node using a whitelist of tag names and attributes

Only elements with names in the whitelist are allowed. Other elements are removed. Those elements can only have attributes in the whitelist of the coressponding tag name. Other attributes are erased.
