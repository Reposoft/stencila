Detailed Description

A where directive (e.g. <div data-where="py,cpp"><span data-text="point.x" /></div> )

Used to restrict the rendering of sections of a stencil to only some context types. This is useful when evaluated expressions are specific to a particular context type. If the current context does not accept one of the labels in the comma separated list the section is turned off.
